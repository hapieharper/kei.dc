# 1주차 학습 허브 (신입용)

이 문서만 보고 순서대로 따라하면 1주차 학습과 보고서 제출까지 완료할 수 있습니다.

## 시작 전

1. `00_START_HERE.md` 먼저 읽고 시작

## 0. 이번 주 최종 목표

1. 보험 청구 데이터 구조를 이해한다.
2. SQL 3개 이상을 직접 실행한다.
3. 보고서 1개를 제출한다.

## 1. 먼저 할 일 (Day 1)

1. `01_온보딩/보험비즈니스_스토리_입문.md` 읽기
2. `01_온보딩/용어집_신입용.md` 읽기

완료 기준:
1. `담보`, `보장`, `면책` 차이를 본인 말로 설명 가능
2. `청구보험금액`, `면책금액`, `결정금액` 관계 설명 가능

## 2. SQL 실습 (Day 2)

### DBeaver 전용 빠른 시작 (필수 경로)

1. `02_SQL실습/SQL_실행환경_가이드.md`의 `DBeaver로 DB 접속하기`만 먼저 수행
2. `02_SQL실습/주요질문_SQL.md`에서 Q1~Q3 실행
3. `02_SQL실습/주요질문_SQL_샘플결과.md`와 핵심 값 비교

### Python/노트북 경로 (선택)

1. `02_SQL실습/SQL_실행환경_가이드.md` 먼저 읽기
2. `02_SQL실습/주요질문_SQL.md` 쿼리 실행
3. `02_SQL실습/주요질문_SQL_샘플결과.md`와 비교

실행 명령:

```powershell
python 1주차/02_SQL실습/scripts/run_sql_workbook.py
```

또는 노트북:

```powershell
jupyter lab
```

노트북 파일:
1. `02_SQL실습/notebooks/week1_sql_eda_practice.ipynb`

완료 기준:
1. SQL 결과 파일(`report.md`, `csv`) 생성 확인
2. 본인이 실행한 SQL 3개를 설명 가능
3. DBeaver 실행 시 Q1~Q3 결과가 샘플결과 문서와 큰 흐름이 일치함
