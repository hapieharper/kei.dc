# 멘토 평가 체크리스트 (질문/실습/보고서)

총점 100점 기준입니다.

## 1) 질문 품질 (30점)

1. 현상 설명 명확성 (10점)
2. 근거 숫자/쿼리 제시 (10점)
3. 가설 또는 선택지 제시 (10점)

## 2) SQL 정확성 (25점)

1. 테이블/컬럼 사용 정확 (10점)
2. 집계 기준(라인수/사고건수) 명시 (10점)
3. 재현 가능성(같은 결과 재실행) (5점)

## 3) 해석력 (25점)

1. 결과 요약 정확성 (10점)
2. 과도한 단정 회피(이상치=오류 단정 금지) (10점)
3. 비즈니스 의미 연결 (5점)

## 4) 실행력/보고서 품질 (20점)

1. 템플릿 충족(목적/SQL/결과/해석/액션) (10점)
2. 후속 액션 실효성 (10점)
