# EDA 추가 칼럼 실습 예시

이 문서는 `결정금액` 예시에 이어, 신입이 같은 방식으로 EDA를 확장하는 실습 가이드입니다.

## 읽는 법(초보용)

각 예시는 같은 순서로 보면 됩니다.

1. 비즈니스 질문: 왜 이 칼럼을 보는지
2. 핵심 SQL: 최소한으로 어떤 쿼리를 돌릴지
3. 샘플 결과: 숫자를 어떻게 읽을지

## 사용 파일

1. SQL 모음: `1주차/02_SQL실습/sql/eda_추가칼럼_예시.sql`
2. 심화(결정금액): `1주차/02_SQL실습/sql/eda_결정금액_예시.sql`

## 예시 1) `청구보험금액(Integer>Currency)`

비즈니스 질문:
1. 고객이 많이 청구하는 대표 금액 구간은 무엇인가?
2. 고액 청구 비중은 얼마나 되는가?

핵심 SQL:
1. 빈도분석 Top 10
2. IQR 기준 이상치 건수(상한 690,000)

샘플 결과(5줄):

| claim_amt | count |
| ---: | ---: |
| 10,000 | 54,318 |
| 300,000 | 48,396 |
| 60,000 | 43,765 |
| 20,000 | 42,589 |
| 30,000 | 40,514 |

이상치 결과:
- `outlier_cnt = 58,191`

초보 해석:
1. 청구가 특정 금액 구간에 반복적으로 몰리는지 확인
2. 고액 청구는 바로 이상으로 보지 말고 담보/질병과 같이 확인

## 예시 2) `질병코드` + `질병명`

비즈니스 질문:
1. 어떤 질병에서 청구가 집중되는가?
2. 월별/분기별로 상위 질병 패턴이 바뀌는가?

핵심 SQL:
1. 질병코드/질병명 빈도 Top 10

샘플 결과(5줄):

| disease_code | disease_name | claim_lines |
| --- | --- | ---: |
| J209 | 급성 기관지염 | 107,966 |
| J304 | 혈관운동성 및 알레르기성 비염 | 32,253 |
| J219 | 급성 세기관지염 | 26,453 |
| J189 | 상세불명 병원체의 폐렴 | 25,858 |
| S134 | 목 부위에서의 관절 및 인대의 탈구, 염좌 및 긴장 | 17,163 |

초보 해석:
1. 상위 질병은 "자주 발생하는 청구 유형"으로 이해
2. 다음 단계에서 해당 질병의 평균 결정금액까지 함께 확인

## 예시 3) `병원종별구분코드`

비즈니스 질문:
1. 어떤 병원 종별에서 청구가 가장 많은가?
2. 종별에 따라 평균 결정금액 차이가 있는가?

핵심 SQL:
1. 병원종별구분코드 분포
2. (추가) 종별별 평균 결정금액 비교

샘플 결과(5줄):

| hospital_type_code | claim_lines |
| --- | ---: |
| C | 360,707 |
| B | 191,554 |
| A | 151,989 |
| M | 82,029 |
| E | 12,654 |

초보 해석:
1. 건수 비중이 큰 병원종별부터 우선 분석
2. 건수와 금액은 별개이므로 평균/중앙값도 같이 확인

## 예시 4) `사고청구년월일` + `지급년월일` (파생: 지연일수)

비즈니스 질문:
1. 청구 후 며칠 내 지급이 끝나는가?
2. 지연일수가 긴 건은 어떤 특성이 있는가(담보/질병/병원)?

핵심 SQL:
1. `lag_days = 지급일 - 청구일` 분포 Top 10
2. 지연일수별 평균 결정금액

샘플 결과(5줄):

| lag_days | count |
| ---: | ---: |
| 0 | 399,964 |
| 1 | 202,740 |
| 3 | 65,824 |
| 2 | 34,116 |
| 4 | 22,928 |

초보 해석:
1. 0~1일 처리 비중이 높으면 운영이 빠른 편
2. 장기 지연 구간은 별도 원인 분석(서류 미비/심사 복잡도/특수 담보)

## 신입에게 보여줄 실습 순서(권장)

1. 숫자형 컬럼 1개(`청구보험금액`)로 빈도/이상치 수행
2. 코드형 컬럼 1개(`질병코드`)로 집중도 파악
3. 분류형 컬럼 1개(`병원종별구분코드`)로 구성비 파악
4. 날짜 2개로 파생변수(`lag_days`)를 만들어 운영지표 분석
5. 마지막에 `질병코드 x 담보코드명 x 고액여부` 교차분석 진행
